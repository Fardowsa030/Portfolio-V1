OVER MIJ

Mijn interesse in software begon al van jongs af aan. Tijdens de intense game sessies met mijn familie en vrienden was ik altijd benieuwd naar hoe de game was gemaakt. Mijn nieuwsgierigheid bleef niet alleen bij games, maar groeide uit naar websites en applicaties. Om mezelf er in te kunnen verdiepen wist ik dus dat ik ICT moest gaan studeren. Momenteel studeer ik HBO:ICT op de Hogeschool Utrecht. Als richting heb ik Software development gekozen, omdat ik geïnteresseerd ben in het creëren van software. Op de Hogeschool Utrecht ontwikkel ik me verder om een betere Software developer te worden, door niet alleen maar op de back-end te focussen, maar ook op de front-end. Uiteindelijk hoop ik dus een Full-stack developer te kunnen worden.
